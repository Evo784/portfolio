Contexte

Un site nécessaire
sous contrainte réelle.

L'ARCT (Association Auto RC Thonon) avait besoin d'un site web rapidement — pas pour des raisons marketing, mais pour un dossier de demande de terrain auprès de la mairie. Délai court, client non-technique, contrainte administrative concrète.

Stack choisi délibérément minimal : HTML/CSS/JS vanilla + PHP uniquement pour le formulaire de contact. Zéro framework, zéro dépendance runtime. Pas par manque de compétence — par pragmatisme : un bénévole non-dev qui hérite du site dans 2 ans doit pouvoir le modifier sans formation.

Ce qui est bien fait

Simple ne veut pas
dire bâclé.

Sur un site statique, la vraie valeur ajoutée est dans les détails : SEO complet, images optimisées, formulaire sécurisé. Ce sont ces détails qui font la différence entre "ça tourne" et "c'est pro".

  • ▸ SEO complet sur 9 pages — balises OG, Twitter cards, URLs canoniques, sitemap.xml, robots.txt
  • ▸ Optimisation images WebP — -88% à -95% sur les fichiers les plus lourds sans perte visuelle
  • ▸ Formulaire PHP sécurisé — honeypot anti-spam, validation serveur, modale async (pas de rechargement)
  • ▸ Performance mesurée — Lighthouse performance 90+ malgré les images photo de voitures RC
  • ▸ Design cohérent avec l'identité "racing" de l'association — sobre mais reconnaissable
  • ▸ Hébergement o2switch configuré : HTTPS, redirections www, headers sécurité
Formulaire de contact

Anti-spam sans librairie.

  • ▸ Honeypot — champ caché CSS, si rempli → le message est silencieusement ignoré
  • ▸ Validation serveur PHP — sanitisation des inputs, vérification format email
  • ▸ Soumission async — fetch + modale de confirmation, pas de rechargement de page
  • ▸ Protection headers PHP — pas d'injection possible dans les en-têtes mail
