Vue d'ensemble

Un site vitrine.
Un admin panel
from scratch.

Ressourcez-vous est le site vitrine d'une esthéticienne indépendante. Le besoin : un site professionnel qu'elle gère seule — modifier ses tarifs, ajouter des catégories de prestations, mettre à jour son équipe — sans passer par moi.

Le CMS choisi est Craft CMS 5 (pas WordPress — admin plus propre, modèles de contenu flexibles, meilleures performances). Mais l'admin natif de Craft est trop technique pour une cliente non-dev. J'ai donc construit un panel d'administration 100% custom par-dessus, accessible sur /gestion — avec son propre design, son propre système d'auth, et une interface pensée pour quelqu'un qui n'est pas développeur.

Architecture du panel admin

Module Craft dédié.
Pas un plugin.
Du code maison.

L'admin est un module Craft PHP (GestionModule) avec un controller dédié (AdminController). Il ne dépend d'aucun plugin du marketplace — chaque fonctionnalité est écrite à la main.

  • ▸ GestionModule — module Craft PHP enregistré au bootstrap, intercepte les requêtes sur /gestion
  • ▸ AdminController — controller unique qui gère toutes les actions CRUD + auth + backups
  • ▸ Templates Twig isolés — layout admin.twig séparé du site public, avec sa propre sidebar et son design system
  • ▸ Stockage JSON — les données métier (prestations, marques, équipe) sont stockées en fichiers JSON, pas en BDD relationnelle — simple, versionnable, backupable
Authentification & sécurité

3 couches d'auth.
0 librairie tierce.

Chaque mécanisme de sécurité est implémenté manuellement en PHP natif. L'authentification est vérifiée à 3 niveaux — si un niveau est contourné, les deux autres bloquent.

  • ▸ Defense in depth — auth vérifiée au niveau Module (EVENT_BEFORE_REQUEST), au niveau Template (admin.twig), et au niveau Controller (requireAuth sur chaque action d'écriture)
  • ▸ Hachage bcrypt — password_hash() / password_verify() PHP natif, mot de passe hashé dans le .env
  • ▸ Tokens CSRF — chaque formulaire protégé via csrfInput() de Craft
  • ▸ Rate limiting from scratch — compteur par IP en fichier JSON, fenêtre glissante de 15 minutes
  • ▸ Sessions sécurisées — sliding window 30 min, régénération d'ID après login (anti session-fixation), timer JS côté client
  • ▸ Reset mot de passe — tokens hashés SHA-256, expiration 1h, envoi par email
  • ▸ Upload sécurisé — validation MIME via finfo PHP natif, limites 2 Mo / 5 Mo selon le type
CRUD & routes dynamiques

3 sections éditables.
Routes auto-générées.

La cliente a un CRUD complet sur trois types de contenu. Quand elle crée une nouvelle catégorie de prestations, une page publique est automatiquement générée avec sa propre route — sans intervention technique.

  • ▸ Prestations & tarifs — CRUD sur les catégories, sections et prestations individuelles (nom + durée + prix). Les nouvelles catégories génèrent automatiquement une page sur le site avec leur route et une image hero
  • ▸ Marques partenaires — nom + logo (upload image), organisées par catégorie (Soins, Parfums, Enfants)
  • ▸ Équipe — nom, poste, spécialité par membre
  • ▸ Périmètre volontairement limité — pas d'accès au blog, aux pages légales, au layout ou au CSS. L'interface est simplifiée au strict nécessaire pour une utilisatrice non-technique
Backups & résilience

Backup automatique.
Restauration 1 clic.

Chaque sauvegarde de contenu déclenche un backup JSON automatique. La cliente peut restaurer n'importe quel état précédent sans assistance.

  • ▸ Backup avant chaque sauvegarde — un snapshot JSON est créé automatiquement pour les prestations, marques et équipe
  • ▸ Rotation automatique — les 10 derniers backups par type sont conservés dans storage/backups-admin/, les plus anciens sont supprimés
  • ▸ Page /gestion/sauvegardes — interface dédiée pour restaurer ou supprimer manuellement les backups
  • ▸ Vérification de type — impossible de restaurer un backup équipe dans les prestations (validation croisée)

Zéro jargon.
Autonomie totale.

  • ▸ Guide d'utilisation intégré — accessible sur /gestion/guide, 7 sections couvrant chaque fonctionnalité + un PDF imprimable
